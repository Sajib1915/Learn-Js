ğŸš€ Getting Started
Prerequisites

A modern web browser (Chrome, Firefox, Edge, Safari)
A code editor (VS Code recommended)
Basic understanding of HTML

Running the Code

Clone this repository:

bash   git clone https://github.com/Sajib1915/Learn-Js.git

Navigate to the project directory:

bash   cd Learn-Js

Open index.html in your browser to run JavaScript files
Or use Node.js to run JavaScript files directly:

bash   node Practise.js

   node operator.js
